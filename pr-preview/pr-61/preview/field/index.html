Field

A labeled form control — label, native input, and hint. Wraps real <input>/<select>/<textarea> so it inherits all native behavior (constraint validation, autofill, IME). State is driven by native pseudos and reflected onto the border/hint via :has() — no class toggles, no JS.

Text input

Focus state

Focus the field below — the border shifts to the focus color and a soft accent glow appears (a border swap alone is below the perceptual threshold).

Invalid state

Type something invalid then tab away — :user-invalid fires on blur, so an empty required field doesn't scream on load.

Disabled

Checkboxes & radios

Native checkbox/radio/range/color inputs inherit the brand accent automatically (set in the base layer) — no custom widgets, no JS. Pair them with a .choice-group for accessible, aligned lists.
