Progress

A linear progress indicator. Determinate (data-value) shows a fill proportional to the value; indeterminate shows an animated sweep. Tone variants via [data-tone].

Determinate

Profile completeness 85%
Storage used 64%
API quota 92%
Onboarding complete 100%

Indeterminate

Use when the task duration is unknown.
