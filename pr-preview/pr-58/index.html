Porchlight

A no-dependency, native-CSS framework for building accessible, themeable web applications.

Pre-1.0, in active development.

CSS

Zero JavaScript

Every component is pure CSS. No runtime, no bundle, no hydration. Behavior (tab switching, toggles) is the app’s job via data attributes.

WCAG

WCAG-AA verified

Color pairs are tested for contrast in both light and dark themes. Forced-colors, reduced-motion, RTL, and 200% zoom are all covered.

@scope

Modern CSS first

@scope, @layer, @property, OKLCH, light-dark(), color-mix(), :has(), container queries, Popover API, anchor positioning. Baseline 2025+.

--pl-*

Token-driven

Override one custom property to retheme the entire system. Primitive scales, semantic pairs, and per-component --c-* tokens.
