Tooltip

A hover/focus tooltip using CSS Anchor Positioning. Zero JavaScript for positioning. Auto-flips when near viewport edges. Falls back to hidden in browsers without anchor support.

Basic

This is a tooltip with helpful context. Tooltips can contain longer text that wraps to multiple lines when the content exceeds the maximum inline size.

On an icon

This metric measures weekly active users.

Keyboard accessible

Tab to the button and focus it to show the tooltip.

Tooltips appear on focus too, for keyboard users.
