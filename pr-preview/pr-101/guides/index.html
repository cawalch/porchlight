Guides

How-to guides for installing, theming, and extending Porchlight.

  • Getting Started — Install Porchlight, import only the CSS you need, and set up the cascade layer for host overrides.
  • Architecture & layer model — How Porchlight is layered, and how host applications override it.
  • Layout & Page Building — Use Porchlight's layout primitives, app shell, and container-query sidebar to compose responsive SaaS pages.
  • Composition Recipes — Model-friendly recipes for composing Porchlight app shells, dashboards, data regions, settings pages, and dense admin views.
  • Theming — Override design tokens, create brand themes, and understand the light-dark() color system.
  • Utilities — A small, finite set of single-purpose helpers for accessibility, layout, and typography. Not a utility-first framework.
  • Browser Support — The feature matrix - what Porchlight requires, what degrades, and what is experimental.
  • Enhancements — Progressive enhancement features gated behind @supports. The framework works without them; they add polish where the browser supports them.
