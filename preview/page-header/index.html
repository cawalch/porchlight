Page header

A bare title + actions row for use inside a container. Unlike .c-toolbar (which has padding, border, and a surface background for shell-level bars), .c-page-header has no chrome — no background, no border, no padding. It eliminates the ~65px of dead space that .c-toolbar creates when nested inside .l-container.
