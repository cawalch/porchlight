# Color Tokens

Porchlight colors are authored in **OKLCH** for perceptual uniformity. All
semantic colors use `light-dark()` so they adapt automatically to the
user's color scheme.

## Brand ramp

A 9-step ramp at hue 260 (blue). Override `--pl-brand-*` to retheme.

## Semantic pairs

Foreground/background are defined as intentional pairs. Each pair is
verified to meet WCAG-AA contrast (4.5:1 minimum) in both light and dark
themes via the deterministic contrast test suite.

| Token | Light value | Dark value | Purpose |
|-------|-------------|------------|---------|
| `--pl-color-bg` | `oklch(98% 0.01 260)` | `oklch(15% 0.02 260)` | Page background |
| `--pl-color-surface` | `oklch(100% 0 0)` | `oklch(19% 0.02 260)` | Cards, panels |
| `--pl-color-surface-2` | `oklch(94% 0.01 260)` | `oklch(23% 0.025 260)` | Hover, subtle surfaces |
| `--pl-color-text` | `oklch(20% 0.02 260)` | `oklch(96% 0.01 260)` | Primary text |
| `--pl-color-text-muted` | `oklch(46% 0.03 260)` | `oklch(75% 0.02 260)` | Secondary text |
| `--pl-color-border` | `oklch(87% 0.02 260)` | `oklch(33% 0.03 260)` | Dividers, borders |
| `--pl-color-accent` | `brand-7` | `brand-4` | Links, primary actions |
| `--pl-color-accent-text` | `white` | `oklch(15% 0.02 260)` | Text on accent |

## Status colors

Three semantic states, each with a fill, background tint, and text color:

| Token group | Light | Dark | Usage |
|-------------|-------|------|-------|
| `--pl-color-danger` | `oklch(58% 0.2 25)` | same | Icon, border |
| `--pl-color-danger-bg` | `oklch(96% 0.03 25)` | `oklch(24% 0.06 25)` | Alert background |
| `--pl-color-danger-text` | `oklch(38% 0.15 25)` | `oklch(90% 0.05 25)` | Text on tinted bg |
| `--pl-color-success` | `oklch(56% 0.16 150)` | same | Icon, border |
| `--pl-color-success-bg` | `oklch(96% 0.03 150)` | `oklch(24% 0.06 150)` | Alert background |
| `--pl-color-success-text` | `oklch(36% 0.13 150)` | `oklch(80% 0.13 150)` | Text on tinted bg |
| `--pl-color-warning` | `oklch(58% 0.17 80)` | same | Icon, border |
| `--pl-color-warning-bg` | `oklch(96% 0.05 80)` | `oklch(26% 0.07 80)` | Alert background |
| `--pl-color-warning-text` | `oklch(42% 0.15 80)` | `oklch(82% 0.13 80)` | Text on tinted bg |
