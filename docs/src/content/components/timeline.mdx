# Timeline

The `.pl-c-timeline` renders a vertical activity feed. Each item has a
tone-colored dot on a connector line, with a header (title + time) and body.

## Semantic HTML

```html
<ol class="pl-c-timeline">
  <li class="pl-c-timeline__item" data-tone="success">
    <span class="pl-c-timeline__dot"></span>
    <div class="pl-c-timeline__content">
      <div class="pl-c-timeline__header">
        <span class="pl-c-timeline__title">Deployed</span>
        <time class="pl-c-timeline__time">2m ago</time>
      </div>
      <p class="pl-c-timeline__body">v2.4.1 deployed.</p>
    </div>
  </li>
</ol>
```

## Tone variants

Set `[data-tone]` on `.pl-c-timeline__item`: `success`, `warning`, `danger`
(default = accent).
