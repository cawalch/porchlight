# Skeleton

The `.pl-c-skeleton` component renders a loading placeholder that mimics the shape
of real content. A CSS-only opacity pulse communicates "loading." Three shapes
via `[data-shape]`: `text` (default), `circle` (avatar), and `rect` (block).

## Semantic HTML

```html
<!-- Text lines -->
<div class="pl-c-skeleton-group">
  <div class="pl-c-skeleton" style="--pl-c-skeleton-w: 40%;"></div>
  <div class="pl-c-skeleton"></div>
  <div class="pl-c-skeleton" style="--pl-c-skeleton-w: 80%;"></div>
</div>

<!-- Avatar + rect -->
<div
  class="pl-c-skeleton"
  data-shape="circle"
  style="--pl-c-skeleton-w: 2.5rem;"
></div>
<div
  class="pl-c-skeleton"
  data-shape="rect"
  style="--pl-c-skeleton-w: 6rem;"
></div>
```

## Reduced motion

Under `prefers-reduced-motion`, the themes layer's universal 1ms clamp stops
the pulse. The skeleton remains as a static grey block - still legible as a
placeholder.
